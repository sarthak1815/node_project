# Assets Folder

This folder is for storing static assets like images, icons, fonts, etc.

Currently, the project uses emoji icons for visual elements, but you can add:
- Car images
- Logo files
- Custom fonts
- Icons
- Other static resources

## Usage

Place your files here and reference them in your components:

```typescript
// In component
imageUrl = 'assets/images/car.jpg';
```

```html
<!-- In template -->
<img src="assets/images/car.jpg" alt="Car">
```
